#Nodejs UDP Tracking

##Requirements
-------------
Docker and Docker Compose
UDP packet sender

##How to 
-------------
Once in project folder :
`docker-compose build`
and
`docker-compose up`

##Configuration 
-------------
Web server: localhost:3000
Udp server: 127.0.0.1:3001

#Use
-------------
###UDP ASCII
{latitude,longitude,time,"A or B", speed}
